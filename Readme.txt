Instructions:
1) Call " make -f makefile all " to compile roman numerical calculator funtions library and test_gui (user interface).
2) Call " make -f makefile test " to compile test dataset.


Notes:
1) The functions are:
	function addition: 	
				Check whether the notation is valid for roman numerical
				Convert them into decimals
				Add them
				Convert them into roman numericals

	function subtraction: 	
				Check whether the notation is valid for roman numerical
				Convert them into decimals
				Subtract them
				Check and convert them into roman numericals

	function addition_nondecimal: 	
				Check whether the notation is valid for roman numerical
				Distribute into baskets 
				Convert them into roman numericals

	function subtraction_nondecimal: 
				Check whether the notation is valid for roman numerical
				Distribute them into 'M','D','C' etc baskets
				Unite them
				Convert baskets into roman numericals

	function roman_num_validity: 	
				Check whether the inputs is correct roman numerical notation 

2) Inside ../test/romancalc_test.check file all the test cases are inserted. New test cases can be added. Test case function is automatically generated by checkmk.
 
3) Inside ../gui/test_gui.c file is provided if any calculation will be made by using user interface.

4) Inside functions messages related with the error case is provided to user.

5) The files inside:
	../include/romancalc.h (headers)
	../src/romancalc.c (source code / functions)
	../test/romancalc_test.check (test function acc. to check unit testing framework)
	../gui/test_gui.c (optional user interface)
 	../makefile
